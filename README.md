# masterMind
Final Project for a Microcontroller course

Implementation of the game Master Mind
Hardware used: Two boards with the MSP430 as microcontroller in addition to several other interfaces including shift registers, buzzers, buttons and LEDs
The project consists of making the boards communicate with each other to coordinate a game of master mind.
Players input guesses into the board, which in turn provides visual and auditive feedback to guide them through the game.
